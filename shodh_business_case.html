Technical Architecture

from shodh_memory import MemorySystem

# Initialize (fully offline)
memory = MemorySystem(storage_path="./robot_memory")

# Record observations
memory.record(
    content="Obstacle at (10,20)",
    experience_type="observation",
    entities=["obstacle_147"]
)

# Retrieve in 2-5ms
results = memory.retrieve(
    query="obstacles nearby",
    max_results=5
)

Defensible Moats

  • Technical Moat: Rust-based edge runtime optimized for constrained hardware (Pi/Jetson). Not trivial to replicate.
